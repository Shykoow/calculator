# Calculator Project

## Description

This is a simple calculator built with HTML, CSS (Tailwind), and JavaScript.
It supports basic arithmetic operations, decimal numbers, and clear/delete functionality.
The calculator is interactive, responsive, and prevents invalid input sequences.

## Features

- Addition (`+`), subtraction (`-`), multiplication (`*`), division (`/`)
- Decimal numbers support (`.`) – only one per number
- Delete last digit (DEL)
- All Clear (AC) – reset input to zero
- Prevents multiple operators in a row
- Error handling for invalid expressions
- Simple and clean UI using Tailwind CSS
